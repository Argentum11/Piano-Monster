# Piano-Monster
Piano-Monster是以模擬鋼琴為基礎，
加入教學、錄影、分享功能的Java專案。

## 使用說明：
| Features               | illustrate      | 
| :-------------: | :--------------------: | 
| 長壓短壓 |  會影響延音的時間。 | 
| 教學模式 | 從選單選擇歌曲，並根據指示操作。 | 
| 分享 | 可直接連接至Twitter或Facebook。 | 
| 錄影 |取名後，可錄製影片（暫無聲音）。 | 

  	 

## 操作按鍵：

白鍵：
鍵盤上的ZXCVBNM,
代表一個八度從低音C到高音C
Do Re Mi Fa So La Si Do

鍵盤上的QWERTYUI
代表一個八度從高音C到更高音C
Do Re Mi Fa So La Si Do

黑鍵：
控制相對位置的黑鍵

鍵盤上的SD GHJ
Do#Re# Fa#So#La#
鍵盤上的23 567
Do#Re# Fa#So#La#

把位：
鍵盤上的方向鍵
左 與 右
可以控制把位的降與升
